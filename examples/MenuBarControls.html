Menu Bar Controls Demo

Standalone HTML example for the HTML to App menu bar bridge: generated icons, optional template rendering, image-plus-title placement, dynamic menu rebuilds, status item clicks, menu callbacks, toggles, nested submenus, badges, background mode, and window title control.
